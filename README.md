## Sobre o Projeto

Este projeto nasceu durante a **Imersão Dev do Futuro** do Dev em Dobro — mas não foi entregue no prazo do evento.

O segundo dia terminou. O certificado chegou. O site, não.

A decisão foi consciente: em vez de forçar uma entrega incompleta, reassisti cada aula, identifiquei cada erro e corrigi um por um. Uma semana depois, o projeto ficou do jeito que deveria ter ficado desde o início.

O resultado é um site de lançamento fictício inspirado no universo visual de *Super Mario Galaxy*, com countdown animado para a "estreia", seção de personagens, trailers e uma identidade visual fiel ao jogo — como se a Nintendo tivesse contratado um estúdio de cinema para produzir o filme.

<br>

## Funcionalidades

- **Countdown animado** — contagem regressiva em tempo real para a data de "estreia" do filme
- **Hero imersivo** — background temático com estrelas e elementos visuais do universo Mario Galaxy
- **Seção de personagens** — apresentação visual dos personagens principais
- **Seção de trailers** — layout cinematográfico para exibição de conteúdo em vídeo
- **Animações com GSAP + ScrollTrigger** — elementos entram em cena conforme o scroll
- **Design responsivo** — experiência consistente do mobile ao desktop
- **Deploy no GitHub Pages** — projeto ao vivo, sem dependências de servidor
